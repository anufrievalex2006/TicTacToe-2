# TicTacToe-2
Усовершенствованная версия классической игры "Крестики-Нолики" с реализованной системой прогресса игрока.

## О проекте
Этот проект является Full-stack приложением для игры в "Крестики-нолики" на одном устройстве с поддержкой локального мультиплеера.
Цель проекта - попрактиковаться в разработке современного приложения на React+TypeScript и интеграции с собственной Backend-частью.

### Возможности игры:
* Авторизация пользователей (логин, регистрация)
* Таблица лидеров с сортировкой
* Система начисления очков за баттлы: +3 за победу, +1 за ничью
* Система рангов (в игре называется "dog"): от 10-го (начинающий) до 1-го (бог)
* Локальный мультиплеер (два игрока играют на одном устройстве)
* Адаптивный и комфортный интерфейс

## Используемые технологии:
**Frontend**:
- React + TypeScript
- React Router
- Mantine UI
- TanStack Query
- Axios
- Zod (валидация)

**Backend**:
- Java + Spring Boot
- PostgreSQL

**Дополнительно**:
- Git

## Инструкция по запуску проекта
### 1. Настроить Backend
1) Клонировать репозиторий:
```bash
git clone https://github.com/anufrievalex2006/TicTacToe-2.git
cd TicTacToe-2/backend
```
2) Настроить базу данных на PostgreSQL, создав таблицу "tictactoe" и указав данные в application.properties
3) Запустить Backend:
```bash
./mvnw spring-boot:run
```
### 2. Настроить Frontend
```bash
cd ..
npm install
npm run dev
```
По итогу приложение будет доступно по адресу http://localhost:5173 (или порт, указанный Vite)
