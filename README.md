# TicTacToe-2
Это усовершенствованная версия популярной игры "Крестики-Нолики"; я назвал её "TicTacToe v2.0"

## В этой игре есть:
* Авторизация пользователей
* Таблица лидеров
* Система начисления баллов за баттлы: +3 очка за победу, +1 очко за ничью
* Смена ранга игрока в зависимости от количества очков: от 10го (начинающий) до 1го (бог)

*Примечание: эта игра работает локально, поэтому здесь предполагается, что два игрока играют в баттл на одном устройстве.*

## **Используемые технологии**:
- Frontend: React + TypeScript, React Router, Mantine UI, TanStack Query, axios, zod (валидация)
- Backend: Java + Spring Boot
- DB: PostgreSQL
