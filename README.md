# TicTacToe-2
Усовершенствованная версия классической игры "Крестики-Нолики" с реализованной системой прогресса игрока.

## О проекте
Этот проект является Full-stack приложением для игры в "Крестики-нолики" на одном устройстве с поддержкой локального мультиплеера.
Цель проекта - попрактиковаться в разработке современного приложения на React+TypeScript и интеграции с собственной Backend-частью.

### Возможности игры:
- Авторизация пользователей (логин, регистрация)
- Таблица лидеров с сортировкой
- Система начисления очков за баттлы: +3 за победу, +1 за ничью
- Система рангов (в игре называется "dog"): от 10-го (начинающий) до 1-го (бог)
- Локальный мультиплеер (два игрока играют на одном устройстве)
- Адаптивный и комфортный интерфейс

## Используемые технологии:
**Frontend**:
- React + TypeScript
- React Router
- Mantine UI
- TanStack Query
- Axios
- Zod (валидация)

**Backend**:
- Java + Spring Boot
- PostgreSQL

**Дополнительно**:
- Git

## Инструкция по запуску проекта
### 1. Клонировать репозиторий
```bash
git clone https://github.com/anufrievalex2006/TicTacToe-2.git
cd TicTacToe-2
```
### 2. Настроить Backend
1) Создайте базу данных "tictactoe" в PostgreSQL
2) Настройте подключение к БД в src/main/resources/application.properties
3) Запустите Backend. Можно через IntelliJ IDEA, а можно через командную строку:
```bash
cd backend
mvnw spring-boot:run
```
Backend будет доступен по адресу http://localhost:8080
### 3. Настроить Frontend
*Примечание: если в командной строке Вы находитесь в папке backend, выйдите из нее, используя **cd ..***
```bash
npm install
npm run dev
```
Приложение будет доступно по адресу http://localhost:5173 (или порт, указанный Vite)
